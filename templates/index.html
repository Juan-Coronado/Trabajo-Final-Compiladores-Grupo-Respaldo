Consejos de uso

Guia breve para escribir entradas validas.

  • Usa el formato INT[a,b](expresion) dx.
  • La variable permitida es x y el diferencial debe ser dx.
  • Puedes usar constantes como pi y e.
  • Se permite multiplicacion implicita: 2x o 3(x+1).
  • Revisa la tabla de lexemas para encontrar linea y columna cuando haya errores.
